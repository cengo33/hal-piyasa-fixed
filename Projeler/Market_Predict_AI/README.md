# Hal Predict AI 🚀

Mersin Hal piyasası için geliştirilmiş, yapay zeka destekli fiyat tahmin ve analiz platformu.

## Özellikler
- **Gerçek Zamanlı Analiz:** Mevcut piyasa fiyatlarının anlık takibi.
- **AI Tahminleme:** Geçmiş veriler ve dış etkenler (hava durumu, talep vb.) kullanılarak yapılan gelecek fiyat öngörüleri.
- **Akıllı İçgörüler:** Fiyat değişimlerinin arkasındaki temel nedenlerin analizi.
- **Premium UI:** Glassmorphism ve modern dashboard tasarımı.

## Teknoloji Yığını
- React
- Vite
- Recharts (Veri Görselleştirme)
- Framer Motion (Animasyonlar)
- Lucide React (İkonlar)

## Kurulum
```bash
npm install
npm run dev
```
